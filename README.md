# Lost & Found System
A web application to report lost and found items with admin panel.

## Features
- User registration with validation (letters-only username, strong password)
- Admin dashboard to manage users
- Item reporting with image upload
- Search and filter items
- Delete user accounts (admin only)

## Technologies Used
- HTML/CSS/JavaScript
- PHP (Backend)
- MySQL (Database)
- MySQLi (Database connection)

## Default Admin Login
- Username: admin
- Password: Admin@123
